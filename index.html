Project details

Title of the main project: FINDING ELECTRICITY THEFT IN SMART GRID
Description: This system provides the information security of electricity of meters. The goal of this system is identifying all malicious users in a neighbourhood area in a smart grid within the shortest detection time. System adopts a group testing method to locate the malicious users. Two considered inspection strategies the find out the malicious user's electricity, first one is scanning method in which users will be inspected individually. Second one is binary search method by which a specific number of users will be examined as a whole.
Technologies Used: Html, CSS, java script, Asp.net with C#
